Want a trip code for this ride?

Trip codes let you get an upfront price for your ride before you hail a cab at airports, hotels, etc. Instead of the driver running the meter.

How It Works

  1. Press 'Create Trip Code'(below) to get a code for the ride you just setup.
  2. Whenever you're ready to ride, walk to the taxi line at your airport, hotel, etc.
  3. Hop into a cab from one of our partner fleets, show the code to your driver, and be on your way!
