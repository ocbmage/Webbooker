How to ride from {{pickupName}}

Cabs are only available at the {{pickupName}}-it pickup lot near Terminal 1. You can walk there or catch the free LAX-it shuttle by waiting by the green columns.

You can get an upfront price for your ride instead of the driver running the meter by creating a Trip Code.

HOW IT WORKS

  1. Press 'Create Trip Code'(below) to get a code for the ride you just setup.
  2. When you get to {{pickupName}}-it, notify the {{pickupName}} staff that you have a Trip Code.
  3. Hop into either a LA Yellow Cab or United Checker Cab, show the code to your driver, and be on your way!
